# CalSnap 🍱

An AI-powered mobile nutrition tracker built specifically for Indian 
and South Indian food culture.

## What it does
- Snap a photo of your food → instant calorie and macro breakdown
- Built-in database of South Indian dishes with accurate nutrition data
- Personalized meal plans based on your goal — fat loss, muscle gain, 
  or lean bulk
- Tracks daily calories, protein, carbs and fat
- Regional pricing for India and global markets

## Tech Stack
- Flutter (mobile — iOS & Android)
- Python + FastAPI (backend + AI)
- TensorFlow/Keras (food image classifier)
- Supabase (database + auth)
- Razorpay + Stripe (payments)
